## **General**

```yaml
video:
  src: 'video.mp4'          # Video source                (string)
  type: 'mp4'               # Video type                  (string)
  button: true              # Shows a play button         (bool)
  autoplay: false           # Autoplay                    (bool)
  preload: 'auto'           # Preload data                (auto/metadata/none)
  poster: 'image.jpg'       # Poster source               (string)
  muted: true               # Mute player                 (bool)
  loop: true                # Loop the video              (bool)
  nosupport: 'No Support'   # No support message          (string)
```
