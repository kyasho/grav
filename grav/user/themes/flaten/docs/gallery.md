## **General**

```yaml
gallery:
  options:
    center: true                # Center                          (bool)
    loop: true                  # Loop                            (bool)
    dots: false                 # Dot navigation                  (bool)
    nav: false                  # Arrow navigation                (bool)
    autoplay: true              # Autoplay                        (bool)
    timeout: 2700               # Time between animation          (int)
    speed: 1100                 # Animation speed                 (int)

  items:
    - title: Sea                # Title                           (string)
      src: 'sea.jpg'            # Source                          (string)
    - title: Bridge
      src: 'bridge.jpg'
    - title: Lake
      src: 'lake.jpg'
    - title: Paris
      src: 'paris.jpg'
```
